## 二、阿里云极速部署（6步）

### 核心6步速查

1. **创建实例**
   - 地域: 中国香港/新加坡
   - 配置: 2vCPU+2GiB+40GiB ESSD
   - 镜像: OpenClaw stable-2026.02专属镜像

2. **放行端口**
   ```bash
   ssh root@服务器公网IP
   firewall-cmd --add-port=18789/tcp --permanent
   firewall-cmd --add-port=22/tcp --permanent
   firewall-cmd --reload
   ```

3. **配置凭证**（阿里云百炼）
   ```bash
   openclaw config set models.providers.bailian.accessKeyId "你的Access Key ID"
   openclaw config set models.providers.bailian.accessKeySecret "你的Access Key Secret"
   openclaw config set models.providers.bailian.baseUrl "https://dashscope-intl.aliyuncs.com/compatible-mode/v1"
   ```

4. **启动服务**
   ```bash
   systemctl start openclaw
   systemctl enable openclaw
   ```

5. **验证部署**
   ```bash
   openclaw version
   curl http://localhost:18789/api/v1/health
   # 浏览器访问 http://服务器公网IP:18789
   ```

6. **配置加速源**
   ```bash
   openclaw config set clawhub.mirror "https://mirror.aliyun.com/clawhub/"
   ```

## 四、自定义技能开发示例

### 天气查询技能完整实现

#### 步骤1: 创建目录
```bash
cd ~/.openclaw/workspace/skills/
mkdir weather-query && cd weather-query
```

#### 步骤2: SKILL.md
```markdown
# Skill: weather-query
## Metadata
- name: weather-query
- description: 实时查询中国城市天气信息
- user-invocable: true
- emoji: 🌤️
- author: 新手开发者
- version: 1.0.0
- dependencies:
  - python3
  - requests

## Description
通过中国天气网公共API查询国内城市实时天气

## Parameters
| 参数名称 | 类型 | 必选 | 描述 | 示例 |
|----------|------|------|------|------|
| city | string | 是 | 中国城市名称（中文） | 北京、上海 |

## Examples
1. 查看北京的实时天气
2. 帮我查询上海今天的天气状况
3. 天气查询：广州
```

#### 步骤3: Python脚本 (`index.py`)
```python
import requests
import json

def get_weather(city):
    url = f"http://wthrcdn.etouch.cn/weather_mini?city={city}"
    try:
        response = requests.get(url, timeout=10)
        response.raise_for_status()
        data = response.json()
        if data["status"] == 1000:
            weather_info = data["data"]["forecast"][0]
            result = f"""
【{city}实时天气】
日期：{weather_info["date"]}
天气状况：{weather_info["type"]}
温度范围：{weather_info["low"]} - {weather_info["high"]}
风力风向：{weather_info["fengxiang"]} {weather_info["fengli"]}
湿度：{data["data"]["shidu"]}
            """
            return result.strip()
        else:
            return f"查询失败：未找到{city}的天气信息"
    except Exception as e:
        return f"查询失败：{str(e)}"

if __name__ == "__main__":
    import sys
    if len(sys.argv) < 2:
        print("请传入城市名称")
        sys.exit(1)
    city = sys.argv[1]
    print(get_weather(city))
```

## 五、常见问题排查

### 问题1: 服务启动失败

**现象**: `systemctl status openclaw` 显示 inactive (dead)

**排查步骤**:
```bash
# 检查端口占用
lsof -i :18789

# 验证API-Key
openclaw models test

# 检查内存
free -h

# 重启
systemctl restart openclaw
```

### 问题2: 技能调用无响应

**排查步骤**:
```bash
# 检查技能状态
openclaw skills status --name "技能名"

# 检查加速源
openclaw config get clawhub.mirror

# 测试网络
ping www.baidu.com

# 重启技能服务
systemctl restart openclaw-skills
```

### 问题3: 模型调用失败

**原因**: 配额不足/凭证错误/权限缺失

**解决**:
- 登录阿里云百炼控制台检查额度
- 重新配置API-Key
- 确认已开通百炼服务

### 问题4: 海外技能安装超时

**解决**: 必须配置阿里云加速源
```bash
openclaw config set clawhub.mirror "https://mirror.aliyun.com/clawhub/"
```

### 核心诊断命令

```bash
# 查看核心服务日志
journalctl -u openclaw -n 100
```

## 六、关键注意事项

### ⚠️ 安全必做

1. **限制渠道访问** - 防止别人滥用
   ```json5
   channels.whatsapp.allowFrom: ["你的手机号"]
   ```

2. **工具白名单** - 防止Agent乱删文件
   ```json5
   tools.allow: ["file.read", "browser.navigate"]
   ```

3. **沙箱模式** - 默认non-main
   ```json5
   sandbox.mode: "non-main"
   ```

### 💡 优化建议

1. **省Token三件套**:
   - temperature: 0.2 (降低瞎编)
   - reserveTokensFloor: 20000 (自动压缩)
   - heartbeat: "30m" (空闲重置)

2. **加速源配置** - 国内必须
   ```bash
   openclaw config set clawhub.mirror "https://mirror.aliyun.com/clawhub/"
   ```

3. **内存监控** - 至少4GB推荐8GB
