## 一、基础服务管理

### 服务生命周期

```bash
# 启动OpenClaw
openclaw start

# 停止OpenClaw
openclaw stop

# 重启
openclaw restart

# 查看状态
openclaw status

# 查看所有组件状态（生成诊断报告）
openclaw status --all
```

### 服务器环境（systemd）

```bash
# 启动服务
systemctl start openclaw

# 停止服务
systemctl stop openclaw

# 重启
systemctl restart openclaw

# 查看状态
systemctl status openclaw

# 查看日志
journalctl -u openclaw -n 100

# 设置开机自启
systemctl enable openclaw
```

## 二、诊断与排错

### 自动修复

```bash
# 自动检查配置并修复常见问题
openclaw doctor --fix
```

### 日志查看

```bash
# 查看Gateway日志
openclaw logs --component gateway

# 查看Agent日志
openclaw logs --agent <agent-id>

# 查看技能日志
openclaw logs --skill <skill-name>

# 查看最近100行系统日志
journalctl -u openclaw -n 100

# 实时跟踪日志
journalctl -u openclaw -f
```

### 环境检查

```bash
# 检查模块状态
openclaw env check-module model
openclaw env check-module gateway
openclaw env check-module skills

# 完整环境诊断
openclaw doctor
```

### 网络与端口

```bash
# 检查端口占用
lsof -i :18789

# 测试本地服务
curl http://localhost:18789/api/v1/health

# 测试外部访问
curl http://服务器公网IP:18789/api/v1/health
```

## 三、模型管理

### 模型查看与测试

```bash
# 列出所有配置的模型
openclaw models list

# 测试模型连接
openclaw models test

# 测试指定模型
openclaw models test --model moonshot/kimi-k2.5
```

### 模型认证配置

```bash
# 交互式配置（推荐新手）
openclaw models auth setup-token

# 直接设置API Key（适合脚本）
openclaw config set models.providers.moonshot.apiKey "你的密钥"
```

### 模型切换与参数

```bash
# 设置默认模型
openclaw config set agents.defaults.model.primary "moonshot/kimi-k2.5"

# 设置温度（控制随机性）
openclaw config set agents.defaults.models.moonshot/kimi-k2.5.params.temperature 0.2

# 查看当前模型配置
openclaw config get agents.defaults.model
```

## 四、网关与通信渠道

### 渠道管理

```bash
# 列出所有渠道
openclaw channels list

# 添加渠道（如飞书）
openclaw channels add feishu

# 添加Telegram
openclaw channels add telegram

# 移除渠道
openclaw channels remove whatsapp

# 启用渠道
openclaw channels enable --name feishu

# 禁用渠道
openclaw channels disable --name telegram
```

### 渠道配置

```bash
# 配置飞书Webhook
openclaw config set channels.feishu.webhookUrl "https://open.feishu.cn/..."

# 配置WhatsApp允许的手机号
openclaw config set channels.whatsapp.allowFrom "[\"+85212345678\"]"

# 配置Telegram Bot Token
openclaw config set channels.telegram.botToken "你的BotToken"
```

### Gateway管理

```bash
# 启动Gateway（独立）
openclaw gateway start

# 停止Gateway
openclaw gateway stop

# 重启Gateway（配置变更后需重启）
openclaw gateway restart

# 查看Gateway状态
openclaw gateway status
```

## 五、Skills技能系统

### 技能安装与管理

```bash
# 从ClawHub安装技能
npx clawhub@latest install <skill-name>

# 示例：安装文件管理
npx clawhub@latest install file-manager

# 列出已安装技能
openclaw skills list

# 查看技能详情
openclaw skills info --name file-manager

# 启用技能
openclaw skills enable --name file-manager

# 禁用技能
openclaw skills disable --name file-manager

# 卸载技能
openclaw skills uninstall --name file-manager

# 更新所有技能
openclaw skills update --all

# 更新指定技能
openclaw skills update --name file-manager
```

### 自定义技能

```bash
# 技能目录位置
~/.openclaw/workspace/skills/

# 创建新技能目录
mkdir ~/.openclaw/workspace/skills/my-custom-skill
cd ~/.openclaw/workspace/skills/my-custom-skill

# 编写SKILL.md（技能说明书）
# 编写index.js或main.py（技能实现）

# 重新加载所有技能
openclaw skills reload

# 验证技能状态
openclaw skills status --name my-custom-skill
```

## 六、浏览器自动化

### 网页访问与截图

```bash
# 访问网页并提取内容
openclaw browse https://example.com

# 截图
openclaw screenshot https://example.com

# 截图并保存文件
openclaw screenshot https://example.com --output /path/to/file.png

# 带等待时间的截图（确保页面加载完成）
openclaw screenshot https://example.com --wait 5000
```

### 网页交互（高级）

```bash
# 点击元素（需配合selector）
openclaw click --selector "button.submit"

# 填写表单
openclaw type --selector "input[name='email']" --text "user@example.com"

# 执行JavaScript
openclaw evaluate "document.title"
```

## 七、文件与系统操作

### 文件管理

```bash
# 读取文件
openclaw file read /path/to/file.txt

# 写入文件
openclaw file write /path/to/file.txt --content "文件内容"

# 创建目录
openclaw mkdir /path/to/directory

# 列出目录内容
openclaw ls /path/to/directory

# 删除文件（谨慎！）
openclaw rm /path/to/file.txt

# 移动/重命名
openclaw mv /old/path /new/path

# 复制文件
openclaw cp /source /destination
```

### 系统命令执行

```bash
# 执行Shell命令
openclaw exec "ls -la"

# 执行Python脚本
openclaw exec "python3 /path/to/script.py"

# 后台执行（不等待结果）
openclaw exec "npm start" --background
```

## 八、配置管理

### 查看与设置

```bash
# 获取配置项
openclaw config get <key>

# 示例：获取端口
openclaw config get gateway.port

# 设置配置项
openclaw config set <key> <value>

# 示例：设置端口
openclaw config set gateway.port 18789

# 获取所有配置
openclaw config get all

# 重置配置为默认
openclaw config reset <key>
```

### 配置层级

- **全局配置**: `~/.openclaw/openclaw.json`
- **Agent配置**: `config/agent-*.js`
- **环境变量**: `OPENCLAW_*`

## 九、实用技巧

### 快速查看运行状态

```bash
# 一键诊断
openclaw doctor --fix && openclaw status --all

# 查看资源使用
openclaw env check

# 检查网络
ping -c 4 api.moonshot.ai
```

### 安全操作建议

1. **先检查，再操作**:
   ```bash
   openclaw ls /path  # 确认文件列表再删除
   ```

2. **使用沙箱模式**:
   ```bash
   openclaw config set sandbox.mode "non-main"
   ```

3. **限制渠道访问**:
   ```bash
   openclaw config set channels.whatsapp.allowFrom "[\"+85212345678\"]"
   ```
