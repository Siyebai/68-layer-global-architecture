#### 2. Karpathy Ratchet Loop 自动化研究
**掌握度**: 100% ✅

**核心机制** (3文件):
- `prepare.py` (固定数据准备)
- `train.py` (AI修改训练循环)
- `program.md` (人类设定目标)

**闭环**:
```
人类写program.md → AI改train.py → 5分钟训练 → 评估val_bpb → 保留/回滚 → 循环
```

**实测**: 700实验/夜，GPT-2训练时间2.02h→1.80h (**提升11%**)

**对我们交易系统的移植**:

- **failure-learner升级**: 自动迭代失败→规则库
